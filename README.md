# 🗂 Java HashMap Practice — OOP Assessment Style

## Overview
In this exercise you will practice declaring, initializing, and manipulating HashMaps in Java.  
Skeleton code is provided. Your task is to implement the TODOs inside each class.

## Learning Outcomes
- Understand the Map interface and HashMap implementation
- Practice declaring and initializing HashMaps
- Practice manipulating HashMaps with built‑in methods
- Reinforce Encapsulation and OOP design

## Time Limit
2 hours

## Project Structure
hashmap-practice/
├── README.md
├── pom.xml
├── src/main/java/com/practice/HashMapExercises.java
└── src/test/java/com/practice/HashMapExercisesTest.java

## Implementation Steps
1. Declare a HashMap to store student IDs and names.
2. Initialize the HashMap with sample data.
3. Add methods to:
   - Add a student
   - Remove a student
   - Find a student by ID
   - Print all students
4. Use TODO comments inside the class as guidance.
5. Run the JUnit tests until all pass.
